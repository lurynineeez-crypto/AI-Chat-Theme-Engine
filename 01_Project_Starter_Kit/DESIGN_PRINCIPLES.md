## Core Principles

### Build the system, not the screenshot.

Reusable systems are more valuable than isolated visuals.

中文说明：
不要只追求单张效果图，要建立可复用的主题系统。

### Theme is visual language.

Theme provides:
- colors
- illustrations
- decorations
- atmosphere

Components belong to the Engine.

中文说明：
主题负责视觉语言，组件负责结构。

### Flat First.

Flat Design defines structure before Premium Design adds presentation quality.

中文说明：
先保证可生产，再增加高级质感。

### Consistency is a feature.

A unified system is more valuable than many isolated designs.
