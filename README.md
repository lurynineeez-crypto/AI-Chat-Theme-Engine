# AI Chat Theme Engine

> Build the system, not the screenshot.

AI Chat Theme Engine is a production-oriented framework for building reusable AI chat themes.

## Vision
- Separate Theme Language from UI Components.
- Build once, create many Theme Packs.
- Keep Components stable and Themes replaceable.

## Core Workflow
1. Theme Idea
2. Theme Language
3. Flat Master
4. Reference Package
5. Component Mapping
6. Codex Implementation
7. Preview
8. QA
9. Release
