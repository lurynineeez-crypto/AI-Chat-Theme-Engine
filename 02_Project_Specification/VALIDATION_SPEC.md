# VALIDATION_SPEC.md

Version: 0.1.1
Status: Draft

## Purpose

Define Theme Pack validation rules.

中文说明：
确保主题包在进入 Engine 前符合标准。

## Validation

Check:

- Manifest validity
- Required files
- Asset paths
- Compatibility version

## Principle

Fail early.

Invalid Theme Packs should be rejected before runtime.
