# COMPONENT_SPEC.md

Version: 0.1.1
Status: Draft

## Purpose

Define responsibilities of Engine-owned UI components.

中文说明：
组件属于 Engine，不属于 Theme。

## Core Components

- Bubble
- Button
- Input
- Avatar
- Card
- Navigation

## Rules

Components provide structure.
Themes provide visual language.

A Theme may decorate components but must not redesign component behavior or layout.

中文说明：
这是整个项目保持一致性的核心规则。
