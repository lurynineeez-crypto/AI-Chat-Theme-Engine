# THEME_MANIFEST_SPEC.md

Version: 0.1.1
Status: Draft

## Purpose

Define the entry manifest of a Theme Pack.

The manifest describes the Theme identity and references modules.

## Responsibilities

The manifest contains:
- id
- name
- version
- engine compatibility
- module references

中文说明：
Manifest 是 Theme Pack 和 Engine 的入口协议。

It should remain small and stable.
